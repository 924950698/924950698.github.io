2. 创建项目

ng new taskmgr -si --style=scss
不支持-si简写的话： ng new taskmgr --skip-install --style=scss

3. 开发模块

  • 先下载依赖：npm install
  • 创建核心模块： ng g m core
    m: module
规则：如何让 core模块 在系统中只加载一次 ？
解决：反依赖注入 
    SkipSelf() 正常情况会反复引用CoreModule，该次注入避免陷入无限循环。 
    Optional() 第一次加载CoreModule时，如果没有，就构造该函数；
                                        如果有，就继续往下执行；
